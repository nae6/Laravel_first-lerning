# Laravel first Learning Repository

## 概要

このリポジトリは、初めてLaravelで環境構築からブラウザへ画面を表示するまでの方法を学ぶ過程で、実践したコードや設定ファイルを管理するために使用します。

## 学習内容 / プロジェクト一覧

 **1. Docker環境構築** | Nginx,PHP,MySql,phpMyAdminの設定,ビルド ||
 **2. Laravelインストール** | composerの確認、プロジェクト作成、時間設定 ||
 **3. Laravelコーディング規約** | PSR ||
 **4. ディレクトリとファイル構成** | ディレクトリ構成、ファイル構成 ||
 **5. ブラウザへ画面を表示する** | route / controller / view, MVCモデル ||

## 環境構築と実行手順

### 前提条件
* **Docker Desktop** (または Docker Engine) がインストールされていること。
* **Docker Compose** が利用可能であること。
